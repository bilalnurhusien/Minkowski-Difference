
Usage:
	
	$ ./minkowski

Usage:
	./minkowski <num-of-polygon1-verticies> <num-of-polygon2-verticies>
Example:
	./minkowski 6 5 # For a hexagon and a pentagon

	./minkowski -f <input-file> # CSV input file

Library Installation:

	$ sudo apt-get install libsfml-dev

Output: 

	Minkowski sum

	# For two pentagons
	$ ./minkowski 5 5
